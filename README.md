# nsfw-api
an api to "generate" nsfw images randomly. 

# running 
```
git clone https://github.com/Swag666baby/nsfw-api
bash build.sh
```
or
```
git clone https://github.com/Swag666baby/nsfw-api
yarn add express lodash @types/node
npm install -g typescript
tsc app.ts
```
